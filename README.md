# Azkar

A simple and beautiful web app that displays morning and evening azkar (Islamic remembrances).

## Features

*   Displays morning and evening azkar in a clean and easy-to-read format.
*   Beautiful and responsive design that works on all devices.
*   Dark and light modes.
*   Share azkar with your friends and family.

## How to use

1.  Clone the repository.
2.  Open the `index.html` file in your browser.
